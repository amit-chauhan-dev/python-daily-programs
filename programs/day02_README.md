# Day 02 - Arithmetical Operations (Addition and Division)
## Description
This Python program performs basic arithmetical operations:
- Addition of two numbers
- Division of two numbers

The program also checks if the divisor is zero to prevent division errors.

## Concepts Used
- Python input() function
- float() type conversion
- Arithmetic operators (+, /)
- Conditional statements (if-else)
- f-strings for formatted output

## Program Flow
1. User enters two numbers for addition.
2. Program calculates and displays the sum.
3. User enters two numbers for division.
4. Program checks if the divisor is zero.
5. If not zero, the division result is displayed.

## Example Output

Addition:
Enter the first number for addition: 5  
Enter the second number for addition: 6  
Sum: 5.0 + 6.0 = 11.0

Division:
Enter the dividend for division: 25  
Enter the divisor for division: 5  
Division: 25.0 / 5.0 = 5.0
