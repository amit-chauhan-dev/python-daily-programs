## 📌 Problem
Write a Python program to generate a random number within a user-defined range.

## Program Description
This Python program generates a random number between two numbers entered by the user.

The program uses Python's built-in `random` module and the `randint()` function to generate a random integer within the specified range.

Steps performed in the program:
1. Import the `random` module.
2. Take the starting number from the user.
3. Take the ending number from the user.
4. Generate a random number between the given range.
5. Display the generated random number.

## Example Output

Random Number Generator

Enter the starting number: 1  
Enter the ending number: 100  

Generated random number: 57

(Note: The result will be different every time the program runs.)

## Concepts Used
- Python modules
- `import` statement
- `random` module
- `randint()` function
- User input (`input()`)
- Integer conversion (`int()`)
