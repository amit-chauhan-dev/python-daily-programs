## 📌 Problem
Write a Python program to convert distance from kilometers to miles.

## Description
This Python program takes distance in kilometers as input from the user and converts it into miles using the standard conversion factor.

## Concepts Used

- Python input()
- Float data type
- Arithmetic operations
- Variables
- f-string formatting

## Formula

1 kilometer = 0.621371 miles

Miles = Kilometers × 0.621371
