## Problem

Write a Python program to check whether a number is positive, negative, or zero.

---

## Program Description

This Python program checks the nature of a number entered by the user.

The program:

1. Takes a number as input from the user.
2. Compares the number with zero.
3. Determines whether the number is:

   * Positive
   * Negative
   * Zero
4. Displays the result.

---

## Formula Used

* If number > 0 → Positive
* If number = 0 → Zero
* If number < 0 → Negative

---

## Concepts Used

* Python input()
* Float data type
* Conditional statements (`if`, `elif`, `else`)
* Comparison operators
