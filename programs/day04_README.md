## 📌 Problem
Write a Python program to swap the values of two variables.

## Program Description
This Python program swaps the values of two variables using a temporary variable.

Steps performed in the program:
1. The program takes two inputs from the user.
2. It displays the original values of the variables.
3. A temporary variable is used to store one value.
4. The values are swapped.
5. The swapped values are displayed.

## Example Output

Enter the value of the first variable (a): 10  
Enter the value of the second variable (b): 20  

Original values: a = 10, b = 20  
Swapped values: a = 20, b = 10
