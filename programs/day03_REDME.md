## Write a Python program to find the area of a triangle

## 📌 Problem
Write a Python program to calculate the area of a triangle using the base and height provided by the user.

## Formula Used
Area of Triangle = 0.5 × Base × Height

## Program Description
This Python program:
1. Takes the base of the triangle as input.
2. Takes the height of the triangle as input.
3. Calculates the area using the formula.
4. Displays the result to the user.

## Example Output

Enter the length of the base of the triangle: 10
Enter the height of the triangle: 5

The area of the triangle is: 25.0
