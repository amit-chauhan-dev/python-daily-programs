## Problem
Write a Python program to display the calendar of a specific month and year.

## Program Description
This Python program uses Python's built-in `calendar` module to display the calendar of a given month and year.

The user enters the year and month, and the program prints the formatted calendar for that month.

## Example Output

Enter the month: 3

## Concepts Used

- Python modules
- `import` statement
- `calendar` module
- User input (`input()`)
- Integer conversion (`int()`)
